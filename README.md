# Movie Search Application

## Description
A movie search application that allows users to search for movies and view details using an external API.

## Tech Stack
- React.js
- JavaScript
- API Integration
- HTML
- CSS

## Features 
- Search movies by name
- Display movie posters and details
- Dynamic API-based results
- Responsive interface

## How to Run
1. Clone the repository
2. Add your API key in the environment file
3. Run `npm install`
4. Run `npm start`
